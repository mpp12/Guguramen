Pokedex v1.0

A Pokedex programmed with Python. I used the tkinter module for the UI and a Pokemon API to obtain data for the pokemon in a JSON format. Top-100-song-search

This Python program takes a date as an input and returns the billboard 100 songs for that date. The regex module is used to validate the user date input. Beautifulsoup was used to scrape billboard.com for the top 100 songs.
